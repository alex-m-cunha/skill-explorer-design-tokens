## v1.0 — 2026-05-25

### Added

- Initial public documentation page for Skill Explorer Design Tokens v1.0.
- Base CSS custom properties for colors, typography, spacing, radius, and separators.
- Documentation structure for Brightspace Catalog and LMS implementation.
- Examples for Video Intro and Reading templates.
- Placeholder system for images, videos, charts, and documents.
- GitHub Pages-ready static HTML structure.

### Defined

- `#2C2C2C` as primary black.
- `#FFFFFF` as base white.
- `#009BD0` as official Nova Executive brand color.
- `#0077A3` as accessible link / interaction color.
- Playfair Display for editorial and narrative headings.
- Open Sans for functional headings, body copy, micro-copy, captions, instructions, labels, and interface text.
- `400` as normal reading weight.
- `600` as heading, label, and emphasis weight.
- Theme customization layer for one primary color per client/cohort.
- Asset requirements for LMS cover images and Unit of Learning banners.

### Notes

- The system is intentionally lightweight.
- Logos and images are documented as implementation assets, not core tokens.
- Client/cohort customization should not change the underlying template structure or accessibility rules.
- Future changes should be documented when they affect token values, naming, accessibility, template logic, or Brightspace implementation guidance.
